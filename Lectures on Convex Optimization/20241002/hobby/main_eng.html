The Pitfall of the Statement "A Convex Function is Continuous"

A convex function defined on an open interval is continuous. In this sense, it can be said that "a convex function is continuous."

However, a convex function defined on a closed interval is not necessarily continuous at the endpoints of the interval.

If you assume that the function is continuous as long as it is not discontinuous at the endpoints, you may fall into a trap, as illustrated by the following example:

Continuity

Though it may not be explicitly described in textbooks, continuity can generally be defined as follows without any issues. For further details, please refer to sources like this one.

The fact that a function fff is continuous on dom f\mathrm{dom} ~ fdom f is equivalent to fff being continuous at any x‾∈dom f\overline{x} \in \mathrm{dom} ~ fx∈dom f.

For a specific x‾∈dom f\overline{x} \in \mathrm{dom} ~ fx∈dom f, this is defined as follows:

For any sequence {xk}⊆dom f\{ x_k \} \subseteq \mathrm{dom} ~ f{xk​}⊆dom f converging to x‾\overline{x}x, the sequence {f(xk)}\{ f(x_k) \}{f(xk​)} converges to f(x‾)f(\overline{x})f(x).

A Closed Convex Function is Lower Semicontinuous

In the counterexample above, we demonstrated that a closed convex function is not necessarily continuous.

However, the weaker assertion that a closed convex function is always lower semicontinuous holds (see textbook Theorem 3.1.4.1).

(Note that although omitted in this article, the necessary and sufficient condition for a function to be a strictly convex and closed convex function is that it is lower semicontinuous. Also, the example in the second figure of this article is a convex function but neither closed convex nor lower semicontinuous.)

In this section, I will present the definition of lower semicontinuity and then show that a closed convex function is lower semicontinuous.

Definition of Lower Semicontinuity

Lower semicontinuity, like continuity, can be defined as follows:

For any sequence {xk}⊆dom f\{ x_k \} \subseteq \mathrm{dom} ~ f{xk​}⊆dom f converging to x‾\overline{x}x, lim inf⁡k→∞f(xk)≥f(x‾)\liminf_{k \to \infty} f(x_k) \geq f(\overline{x})liminfk→∞​f(xk​)≥f(x).

Indeed, in the previous example, the sequence of red points converging to (g‾,γ‾)=(0,0)(\overline{g},\overline{\gamma})=(0,0)(g​,γ​)=(0,0) also shows that the function value ψ\psiψ converges to some β>0\beta>0β>0, satisfying lim inf⁡k→∞ψ(gk,γk)=β>ψ(0,0)=0\liminf_{k \to \infty} \psi(g_k,\gamma_k) = \beta > \psi(0,0) = 0liminfk→∞​ψ(gk​,γk​)=β>ψ(0,0)=0.

Convex Functions on Open Intervals are Continuous

Next, we will show that convex functions defined on open intervals are continuous. This fact can be proven quite simply and elementarily, but we will omit such a proof in this section. Please refer to another source for details.

More precisely, in this section, we will prove the following proposition (Textbook Lemma 3.1.4):

A closed convex function of one variable is continuous on dom f\mathrm{dom} ~ fdom f.

This proposition complements the previously proven fact that

A closed convex function of two or more variables is not necessarily continuous on dom f\mathrm{dom} ~ fdom f.

This proposition establishes the continuity of convex functions without discontinuities on closed intervals, and as a corollary, the continuity of convex functions on open intervals immediately follows.

Thus, the following is continuous:

Even as one endpoint approaches infinity, the function remains continuous.

We will now demonstrate that a closed convex function of one variable is continuous on dom f\mathrm{dom} ~ fdom f.

We focus on the question of why, in dimensions higher than one, we can only claim lower semicontinuity, whereas in one dimension, we can claim continuity, as we proceed with the proof.

As a supplement, let’s briefly consider why the above proof does not hold when dealing with more than one variable.

For the function ψ\psiψ, which is a closed convex function but not continuous, its domain is dom ψ=(R×{γ>0})∪{(0,0)}\mathrm{dom} ~ \psi = (\mathbb{R} \times \{ \gamma > 0 \}) \cup \{ (0,0) \}dom ψ=(R×{γ>0})∪{(0,0)}. Sufficient condition for "convex functions are continuous"

To conclude this article, I will present a sufficient condition under which convex functions are continuous, a topic that is somewhat outside the scope of standard textbooks. However, the essence of this condition can be found in articles such as this one.

The following proposition holds:

A convex function f:Rn→Rf: \mathbb{R}^n \to \mathbb{R}f:Rn→R is continuous.

You may wonder, "Wasn't the statement 'convex functions are continuous' proven false earlier in this article?" However, this proposition is true.

The key difference lies in the fact that this convex function takes values only in the real numbers, not extended real numbers. In previous discussions, the convex function fff took values in the extended real number system, i.e., f:Rn→R∪{±∞}f: \mathbb{R}^n \to \mathbb{R} \cup \{ \pm\infty \}f:Rn→R∪{±∞}. In other words, the following proposition is equivalent:

A convex function defined over the entire real domain is continuous.

This assumption prevents counterexamples like dom ψ=(R×{γ>0})∪{(0,0)}\mathrm{dom} ~ \psi = (\mathbb{R} \times \{\gamma > 0\}) \cup \{ (0,0) \}dom ψ=(R×{γ>0})∪{(0,0)} from arising.

Note that functions such as f(x)=x2f(x) = x^2f(x)=x2, where sup⁡f(x)=∞\sup f(x) = \inftysupf(x)=∞, still satisfy the conditions of this proposition. Conditions such as "boundedness of fff" are not required.

However, it is important to note that well-known convex functions such as 1/x1/x1/x and −log⁡x-\log x−logx do not satisfy the condition that "dom f\mathrm{dom} ~ fdom f is the entire real line."

The pitfall of the statement "convex functions are continuous" remains, and a proper understanding of the assumptions under which a proposition holds is crucial.
